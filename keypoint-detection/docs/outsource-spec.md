# 衣類キーポイント アノテーション作業 仕様書

## 概要

衣類の写真を撮影し、専用ツールで「キーポイント」（襟・肩・袖口・裾などの位置）をタップで記録する作業です。
AI採寸モデルの学習データとして使用します。

---

- **1枚あたり30円 × 200枚 = 6,000円**
- 撮影 + アノテーション込みの単価です
- 納期は別途ご相談

## お送りするもの

1. **衣類**（撮影後に返送していただきます）
2. **A4コピー用紙**（スケール基準として使用。白紙でOK）
3. **撮影・作業手順書**（この文書）

## 作業の流れ

### ステップ1: 撮影

1. 衣類を**硬い平面**（テーブル・床）に平置きする
2. A4用紙を**衣類の右下に少し離して**置く
3. **真上から**スマホで撮影する

#### 撮影ルール

- **必ず真上から撮る**（斜めNG）
- 0.5x（超広角）で撮ると簡単です
- **衣類全体**が画面に収まること
- A4用紙が画面内に見えること
- 足が映ってもOK（AIが無視します）
- 1枚につき1着

#### カテゴリ別の撮影方向

| カテゴリ | 撮影面 |
|----------|--------|
| トップス / ジャケット / スーツ（上） / ルームウェア（上） / マタニティ | **背面**（裏返して背中側を上にする） |
| ワンピース / ドレス・ブライダル | **背面** |
| パンツ / スーツ（下） / ルームウェア（下） | **正面** |
| スカート | **正面** |
| サロペット | **正面** |

#### 背面撮影の注意

- 襟は着丈に含めない（襟の付け根からが着丈の起点）
- 袖口は袖の上端にドットを合わせる

#### NG例

- 斜めから撮っている
- 衣類が画面からはみ出している
- A4用紙が衣類の上に置かれている（右下に離して置く）
- シワだらけ（軽く伸ばしてから撮影）

---

### ステップ2: A4用紙の4隅登録

写真をアップロードしたら、まず**A4用紙の4隅**をタップします。

1. ツールが「A4用紙の4隅をタップ」と案内します
2. A4用紙の**左上→右上→右下→左下**の順にタップ
3. 4点打ったら**「次へ」**をタップ
4. A4ドットが消え、キーポイント登録に進みます

※ A4用紙のドットは十字マーク（+）で表示されます

---

### ステップ3: キーポイント登録

画面に表示されるガイドの順番通りに、該当する位置をタップします。

#### 操作手順

1. **共有されたリンクからツールを開く**（お名前は初回のみ入力）
2. **カテゴリを選択**（トップス・パンツなど）
   - スーツ・フォーマル、ルームウェア・パジャマは「上」「下」のサブカテゴリを選択
3. **「画像を選択」**をタップして撮影した写真を選ぶ
4. **A4用紙の4隅をタップ**（ステップ2）
5. ガイドの順番通りに**キーポイントをタップ**
6. **実寸を入力**（メジャーで計測した値）
7. 全項目完了後、**「保存して次へ」**をタップ
8. 次の画像を選んで繰り返し

#### ポイントの修正

- 打ったポイントをタップすると選択状態になり、**ドラッグで位置を調整**できます
- ポイント選択時は自動で拡大表示されます
- 空白部分をタップすると確定
- **「1つ戻る」**で直前のポイントを取り消せます
- 写真を間違えた場合は**右上のリロード**で写真を入れ直せます

---

### ステップ4: 実寸入力

キーポイントを打った後、**メジャーで実際に計測した寸法（cm）**を入力します。

- 実寸入力とキーポイント調整は**順不同**です（先に実寸を入れてからポイントを調整してもOK）
- 入力すると**AI計測値との誤差**がリアルタイムで表示されます
- **全ての採寸値は片面表記**です（×2しない。ウエスト・ヒップも片面のまま入力）

## 左右の基準

**画面を見たときの左右です。**
平置きした衣類を見ている状態で、画面上で左に見えるものが「左」です。

---

## 実寸の計測方法

メジャーで計測してください。**全て片面表記**です（×2しません）。

- **肩幅**: 左肩先〜右肩先の直線距離
- **袖丈**: 肩先〜袖口の直線距離
- **裄丈**: 襟中心（背中心）〜袖口の直線距離
- **身幅**: 左脇下〜右脇下の直線距離
- **着丈**: 襟の付け根（背中心）〜裾の中心。襟は含めない
- **ウエスト**: ウエスト左端〜右端の直線距離（片面）
- **ヒップ**: ヒップ左端〜右端の直線距離（片面）
- **総丈**: パンツ=ウエスト上端〜裾 / スカート=ウエスト上端〜裾 / サロペット=肩紐上端〜裾
- **股上**: ウエスト中心〜股
- **股下**: 股〜裾
- **ワタリ**: 股〜太もも外端の直線距離
- **裾幅**: 裾の左端〜右端の直線距離

## 注意事項

- **1着につき1枚**撮影してください
- カテゴリを間違えないようにしてください（ワンピースなのにトップスで打つ等）
- **スーツ・フォーマル、ルームウェア・パジャマ**はツール上で「上」「下」を選択してください
- ポイントは**衣類の端のライン上**を正確にタップしてください（実際の縫い目や端を狙う）
- 保存後の修正はできないため、保存前に必ず確認してください
- 同じ衣類を2回撮影しないでください
- カテゴリは偏りなく、いろいろな種類を撮影してください
- A4用紙は**白紙のコピー用紙**でOKです（印刷不要）
