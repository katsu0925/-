# 撮影ガイド — AI採寸キーポイント学習用データ

## 概要
平置き衣類の写真から採寸キーポイントを自動検出するAIモデルの学習データを作成するため、
以下の仕様に従って衣類の平置き写真を撮影してください。

---

## 1. 必要枚数

お渡しした衣類（約200着）を、以下の仕様に従って**1着ずつ平置き撮影**してください。

---

## 2. 撮影環境

### 2-1. 背景（最重要）

**必須: 硬くて平らな面に置くこと。**

| OK | NG |
|---|---|
| テーブル | カーペット・ラグ |
| フローリング床 | 毛布・布団の上 |
| 段ボール | ベッドの上 |
| デスク | ソファ |

背景色: **白 or グレーの無地**が理想。フローリング（木目）もOK。
柄物の布やごちゃごちゃした背景はNG。

### 2-2. 照明

- **均一な自然光**がベスト（窓際、曇りの日が理想）
- LED照明もOK（影が少ない拡散光）
- **NG**: 直射日光（強い影ができる）、フラッシュ、暗い部屋

### 2-3. カメラ

- スマートフォンのカメラで十分
- **最低解像度: 3000 x 2000px以上**（iPhone標準で十分）
- HDR: OFF推奨
- ズーム: 1x（デジタルズーム不可）

---

## 3. 衣類の置き方

### 3-1. 基本ルール

1. **前身頃を上にする**（ボタン・ロゴが見える面が上）
2. **ボタン・ファスナーは閉じる**
3. **シワを手で伸ばす**（引っ張りすぎない、自然な状態）
4. **衣類全体が画面に収まること**（端が切れない）
5. **撮影者の手・足・影が写り込まないこと**

### 3-2. トップスの置き方

```
         ┌─────┐
    ┌────┤ 襟  ├────┐
    │袖  │     │  袖│    ← 袖は自然に横に広げる
    │    └┬───┬┘    │       （水平〜やや下向き、45度以内）
    └─┐  │身頃│  ┌─┘    ← 袖口が身頃に重ならないこと
      └──┤    ├──┘
         │    │
         └────┘ 裾
```

- 袖: **自然に横に広げる**。無理に水平にしなくてよい
- 袖口: 身頃や裾に重ならないように
- パーカーのフード: 後ろに折り畳むか、上に広げる

### 3-3. パンツの置き方

```
    ┌──────────┐ ウエスト
    │          │
    │          │
    ├────┬─────┤ 股
    │左脚│ 右脚│
    │    │     │
    └────┴─────┘ 裾
```

- ウエストバンドを上に
- 両脚を自然にまっすぐ下に
- 片脚が下に潜り込まないように（重なりNG）
- ファスナー・ボタンは閉じる

### 3-4. スカートの置き方

- ウエストバンドを上に
- 裾を自然に広げる（無理に広げすぎない）
- プリーツスカートはプリーツを自然な状態で

### 3-5. ワンピースの置き方

- トップスと同様に袖を広げる
- 裾まで全体が画面に収まること（長い場合はカメラを離す）
- ウエストのくびれがある場合は自然な形を維持

---

## 4. A4用紙の配置

### 4-1. A4採寸シート

別途お渡しするPDFを**A4サイズで印刷**してください。
（四隅にL字マーカー、中央に十字線が印刷されています）

### 4-2. 置く位置

**重要: A4用紙は衣類の上（中央）に置く**

```
    ┌────────────┐
    │            │
    │   ┌────┐  │
    │   │ A4 │  │  ← 衣類の中央に置く
    │   │用紙│  │    十字線がカメラの中心目印になる
    │   └────┘  │
    │            │
    └────────────┘
```

- A4用紙の**中央の十字線**をカメラの中心に合わせて撮影（→ 真上から撮れているか確認できる）
- A4用紙の4角のL字マーカーが全て見えること
- A4用紙が折れない・歪まないこと

**NG例:**
- A4用紙を衣類の外（横）に置く → 十字線でカメラ中心を合わせられない
- A4用紙がフレーム外 → スケール算出不能

---

## 5. 撮影方法

### 5-1. 角度

**真上（90度）から撮影する。**

- A4用紙の**十字線にカメラの中心を合わせる** → 真上から撮れているか確認できる
- スマホのカメラ設定で**グリッド線をON**にすると中心が分かりやすい
  - iPhone: 設定 → カメラ → 「グリッド」ON
  - Android: カメラ設定 → 「グリッドライン」ON

### 5-2. 簡単に真上から撮るコツ

**衣類をまたぐように立ち、0.5x（超広角）で見下ろすのが最も簡単です。**

- カメラを**0.5x（超広角）**に切り替えると、腕を高く上げなくても全体が収まる
- 足が映ってもOK（AIが自動で無視します）
- 0.5xがない機種は1xのまま、少し離れて撮影

### 5-3. 距離

- 衣類全体 + A4用紙が画面に収まる距離
- 上下左右に**衣類の外側に10cm以上の余白**があること
- 近すぎて端が切れるのはNG

### 5-3. チェックリスト（撮影前に確認）

- [ ] 衣類は硬い平面の上に置いたか？
- [ ] 前身頃が上か？
- [ ] ボタン・ファスナーは閉じたか？
- [ ] 袖は自然に広げたか？（トップス・ワンピース）
- [ ] シワを手で伸ばしたか？
- [ ] A4用紙は衣類の上（中央）に置いたか？
- [ ] A4用紙の4角が見えるか？
- [ ] 衣類全体が画面に収まっているか？
- [ ] 真上から撮っているか？
- [ ] 手・足・影が写り込んでいないか？
- [ ] 照明は均一か？（強い影がないか）

## 6. 実寸計測

撮影した全200着について、**メジャーで実寸を計測**してスプレッドシートに記録してください。

### 6-1. 計測方法
- **平置きの状態のまま**メジャーで測る（着用して測らない）
- **片面の実寸**を記録する（×2しない）
- cm単位、小数点1桁まで（例: 51.5cm）

### 6-2. 計測項目（カテゴリ別）

**トップス・ジャケット・パーカー等:**
| 項目 | 測り方 |
|---|---|
| 肩幅 | 左右の肩の縫い目の端〜端（水平） |
| 身幅 | 脇の下（袖の付け根）の左右幅 |
| 着丈 | 後ろ襟の中心付け根〜裾の最下端（垂直） |
| 袖丈 | 肩先〜袖口 |

**パンツ:**
| 項目 | 測り方 |
|---|---|
| ウエスト | ウエストバンド上端の左右幅 |
| 総丈 | ウエスト上端〜裾の最下端（垂直） |
| 股上 | ウエスト上端〜股の交差点（垂直） |
| 股下 | 股の交差点〜裾 |
| ワタリ | 股の交差点と同じ高さでの幅 |
| 裾幅 | 裾の左端〜右端 |

**スカート:**
| 項目 | 測り方 |
|---|---|
| ウエスト | ウエストバンド上端の左右幅 |
| ヒップ | ウエストから約18cm下の左右幅 |
| 総丈 | ウエスト上端〜裾の最下端（垂直） |

**ワンピース:**
| 項目 | 測り方 |
|---|---|
| 肩幅 | 左右の肩の縫い目の端〜端 |
| 身幅 | 脇の下の左右幅 |
| 着丈 | 後ろ襟の中心付け根〜裾（垂直） |
| 袖丈 | 肩先〜袖口 |
| ウエスト | くびれ/ゴムシャーリング位置の左右幅 |

### 6-3. スプレッドシートの記入例

| ファイル名 | カテゴリ | 肩幅 | 身幅 | 着丈 | 袖丈 | ウエスト | 総丈 | 股上 | 股下 | ワタリ | 裾幅 | ヒップ |
|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 001.jpg | トップス | 57.0 | 51.0 | 49.0 | 50.0 | - | - | - | - | - | - | - |
| 002.jpg | パンツ | - | - | - | - | 33.0 | 93.0 | 28.0 | 68.0 | 27.0 | 16.5 | - |

- 該当しない項目は「-」と記入
- ゴムウエストは**伸ばさない自然な状態**で測る

## 7. キーポイントアノテーション（CVAT）

撮影した画像に対して、衣類の特徴点（キーポイント）をマーキングする作業です。

### 7-1. CVATのセットアップ

1. **https://app.cvat.ai/** にアクセス → アカウント作成（無料）
2. 「Projects」→「+」→ 新規プロジェクト作成
3. プロジェクト名: 「写メジャー学習データ」
4. Labels に以下のキーポイントテンプレートを作成（別途お渡しするマニュアル参照）
5. 「Tasks」→「+」→ 画像をアップロード

### 7-2. 作業手順

1. 画像を開く
2. 衣類のカテゴリを確認
3. アノテーションマニュアルに従い、各キーポイントを**順番通りに**クリック
4. 全点マーキング後、次の画像へ
5. 全画像完了後 →「Menu」→「Export task dataset」→ **COCO Keypoints 1.0** でエクスポート

### 7-3. 品質チェック

- 全キーポイントをマーキングしたか
- 肩先は袖の途中ではなく、肩の縫い目の端にあるか
- 脇下は衣類のくびれ（袖→身頃の境目）にあるか

詳細はアノテーションマニュアルPDFを参照してください。

---

## 8. ファイル命名規則

連番でOKです。　例: 001.jpg, 002.jpg, 003.jpg ...
ファイル名に特別なルールはありません。重複しなければ大丈夫です。

---

## 9. 納品物（3点）

| # | 納品物 | 形式 |
|---|---|---|
| 1 | **撮影画像** 200枚 | JPEG（3000x2000px以上） |
| 2 | **実寸データ** 200枚分 | スプレッドシート（Googleスプレッドシート or Excel） |
| 3 | **キーポイントアノテーション** | COCO Keypoints 1.0 形式（CVATからエクスポート） |

納品方法: Google Driveの共有フォルダにアップロード。フォルダ分けは不要（1フォルダにまとめてOK）。
- フォルダ構成:
  ```
  撮影データ/
  ├── tops/
  ├── pants/
  ├── skirt/
  └── dress/
  ```

## 8. NG例とOK例

### NG: カーペットの上
衣類が沈み、シルエットが不明瞭になる。縫い目の陰影も消える。

### NG: A4用紙が衣類の上
肩・脇の縫い目が隠れる。衣類の形が紙の重みで変わる。

### NG: 斜め撮影
台形歪みが発生。上下で縮尺が変わり、採寸精度が下がる。

### NG: 袖が身頃に重なる
袖口のキーポイントが見えない。脇下の位置が特定できない。

### OK: 硬い平面 + 真上 + A4横置き + 自然に広げた状態
