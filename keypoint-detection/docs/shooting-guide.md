# 撮影ガイド — AI採寸キーポイント学習用データ

## 概要
平置き衣類の写真から採寸キーポイントを自動検出するAIモデルの学習データを作成するため、
以下の仕様に従って衣類の平置き写真を撮影してください。

---

## 1. 必要枚数

**合計200枚**の平置き衣類写真を撮影してください。

できるだけ**いろいろな種類の衣類**を含めてください（偏りがないように）。
例: トップス、ジャケット・アウター、パンツ、スカート、ワンピース、スーツ、サロペット、ルームウェア等

サイズはS〜XXLまで混在OK。レディース・メンズ混在OK。
**キッズ（子供服）は不要です。**

---

## 2. 撮影環境

### 2-1. 背景（最重要）

**必須: 硬くて平らな面に置くこと。**

| OK | NG |
|---|---|
| テーブル | カーペット・ラグ |
| フローリング床 | 毛布・布団の上 |
| 段ボール | ベッドの上 |
| デスク | ソファ |

背景色: **白 or グレーの無地**が理想。フローリング（木目）もOK。
柄物の布やごちゃごちゃした背景はNG。

### 2-2. 照明

- **均一な自然光**がベスト（窓際、曇りの日が理想）
- LED照明もOK（影が少ない拡散光）
- **NG**: 直射日光（強い影ができる）、フラッシュ、暗い部屋

### 2-3. カメラ

- スマートフォンのカメラで十分
- **最低解像度: 3000 x 2000px以上**（iPhone標準で十分）
- HDR: OFF推奨
- ズーム: 1x（デジタルズーム不可）

---

## 3. 衣類の置き方

### 3-1. 基本ルール

1. **前身頃を上にする**（ボタン・ロゴが見える面が上）
2. **ボタン・ファスナーは閉じる**
3. **シワを手で伸ばす**（引っ張りすぎない、自然な状態）
4. **衣類全体が画面に収まること**（端が切れない）
5. **撮影者の手・足・影が写り込まないこと**

### 3-2. トップスの置き方

```
         ┌─────┐
    ┌────┤ 襟  ├────┐
    │袖  │     │  袖│    ← 袖は自然に横に広げる
    │    └┬───┬┘    │       （水平〜やや下向き、45度以内）
    └─┐  │身頃│  ┌─┘    ← 袖口が身頃に重ならないこと
      └──┤    ├──┘
         │    │
         └────┘ 裾
```

- 袖: **自然に横に広げる**。無理に水平にしなくてよい
- 袖口: 身頃や裾に重ならないように
- パーカーのフード: 後ろに折り畳むか、上に広げる

### 3-3. パンツの置き方

```
    ┌──────────┐ ウエスト
    │          │
    │          │
    ├────┬─────┤ 股
    │左脚│ 右脚│
    │    │     │
    └────┴─────┘ 裾
```

- ウエストバンドを上に
- 両脚を自然にまっすぐ下に
- 片脚が下に潜り込まないように（重なりNG）
- ファスナー・ボタンは閉じる

### 3-4. スカートの置き方

- ウエストバンドを上に
- 裾を自然に広げる（無理に広げすぎない）
- プリーツスカートはプリーツを自然な状態で

### 3-5. ワンピースの置き方

- トップスと同様に袖を広げる
- 裾まで全体が画面に収まること（長い場合はカメラを離す）
- ウエストのくびれがある場合は自然な形を維持

---

## 4. A4用紙の配置

### 4-1. A4採寸シート

別途お渡しするPDFを**A4サイズで印刷**してください。
（四隅にL字マーカー、中央に十字線が印刷されています）

### 4-2. 置く位置

**重要: A4用紙は衣類の上（中央）に置く**

```
    ┌────────────┐
    │            │
    │   ┌────┐  │
    │   │ A4 │  │  ← 衣類の中央に置く
    │   │用紙│  │    十字線がカメラの中心目印になる
    │   └────┘  │
    │            │
    └────────────┘
```

- A4用紙の**中央の十字線**をカメラの中心に合わせて撮影（→ 真上から撮れているか確認できる）
- A4用紙の4角のL字マーカーが全て見えること
- A4用紙が折れない・歪まないこと

**NG例:**
- A4用紙を衣類の外（横）に置く → 十字線でカメラ中心を合わせられない
- A4用紙がフレーム外 → スケール算出不能

---

## 5. 撮影方法

### 5-1. 角度

**真上（90度）から撮影する。**

- A4用紙の**十字線にカメラの中心を合わせる** → 真上から撮れているか確認できる
- スマホのカメラ設定で**グリッド線をON**にすると中心が分かりやすい
  - iPhone: 設定 → カメラ → 「グリッド」ON
  - Android: カメラ設定 → 「グリッドライン」ON

### 5-2. 簡単に真上から撮るコツ

**衣類をまたぐように立ち、0.5x（超広角）で見下ろすのが最も簡単です。**

- カメラを**0.5x（超広角）**に切り替えると、腕を高く上げなくても全体が収まる
- 足が映ってもOK（AIが自動で無視します）
- 0.5xがない機種は1xのまま、少し離れて撮影

### 5-3. 距離

- 衣類全体 + A4用紙が画面に収まる距離
- 上下左右に**衣類の外側に10cm以上の余白**があること
- 近すぎて端が切れるのはNG

### 5-3. チェックリスト（撮影前に確認）

- [ ] 衣類は硬い平面の上に置いたか？
- [ ] 前身頃が上か？
- [ ] ボタン・ファスナーは閉じたか？
- [ ] 袖は自然に広げたか？（トップス・ワンピース）
- [ ] シワを手で伸ばしたか？
- [ ] A4用紙は衣類の上（中央）に置いたか？
- [ ] A4用紙の4角が見えるか？
- [ ] 衣類全体が画面に収まっているか？
- [ ] 真上から撮っているか？
- [ ] 手・足・影が写り込んでいないか？
- [ ] 照明は均一か？（強い影がないか）

## 6. ファイル命名規則

```
{カテゴリ}_{連番3桁}.jpg

例:
tops_001.jpg
tops_002.jpg
...
pants_001.jpg
skirt_001.jpg
dress_001.jpg
```

### カテゴリコード
| カテゴリ | コード |
|---|---|
| トップス（Tシャツ/スウェット/シャツ/パーカー） | tops |
| パンツ（ジーンズ/スラックス/ショートパンツ） | pants |
| スカート | skirt |
| ワンピース | dress |

---

## 7. 納品形式

- 画像フォーマット: **JPEG**（圧縮品質80以上）
- 解像度: 最低3000x2000px
- ファイル命名: 上記ルールに従う
- 納品方法: Google Driveの共有フォルダにアップロード
- フォルダ構成:
  ```
  撮影データ/
  ├── tops/
  ├── pants/
  ├── skirt/
  └── dress/
  ```

## 8. NG例とOK例

### NG: カーペットの上
衣類が沈み、シルエットが不明瞭になる。縫い目の陰影も消える。

### NG: A4用紙が衣類の上
肩・脇の縫い目が隠れる。衣類の形が紙の重みで変わる。

### NG: 斜め撮影
台形歪みが発生。上下で縮尺が変わり、採寸精度が下がる。

### NG: 袖が身頃に重なる
袖口のキーポイントが見えない。脇下の位置が特定できない。

### OK: 硬い平面 + 真上 + A4横置き + 自然に広げた状態
