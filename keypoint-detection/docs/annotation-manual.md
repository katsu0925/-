# アノテーションマニュアル — 衣類キーポイント

## 概要
平置き衣類の写真に対して、採寸に必要なキーポイント（特徴点）をマーキングする作業です。
各キーポイントは衣類の構造的な特徴点（肩先、脇下、裾など）に対応しています。

---

## 1. ツール

**CVAT (Computer Vision Annotation Tool)** を使用します。
- URL: （別途共有）
- エクスポート形式: COCO Keypoint 1.0

---

## 2. カテゴリ別キーポイント定義

### 2-1. トップス (tops) — 10点

```
          [0] 襟中心
     [1]肩先─────────[2]肩先
      │  ＼           ／  │
      │  [3]脇下──[4]脇下 │
      │      │      │     │
   [5]袖口   │      │  [6]袖口
          [7]裾左──[8]裾右
              [9]裾中心
```

| ID | 名前 | 日本語 | 位置の定義 |
|---|---|---|---|
| 0 | collar_center | 襟中心 | 襟ぐりの上端中央（後ろ襟の中心付け根）。丸首なら襟リブの最上部中央 |
| 1 | left_shoulder | 左肩先 | **左の肩の縫い目の最も外側の端**。肩線（襟から肩に向かう縫い目）の終点。ドロップショルダーの場合は肩縫いの端（通常より低い位置にある） |
| 2 | right_shoulder | 右肩先 | 右側の同上 |
| 3 | left_armpit | 左脇下 | **袖と身頃の縫い合わせの最も内側の点**。衣類の外側輪郭を上から下に追ったとき、袖の張り出しが終わって身頃に戻る「くびれ」の点 |
| 4 | right_armpit | 右脇下 | 右側の同上 |
| 5 | left_cuff | 左袖口 | 左袖の末端の中央（袖口リブの最も外側の中央点） |
| 6 | right_cuff | 右袖口 | 右側の同上 |
| 7 | hem_left | 裾左端 | 身頃の裾ラインの最も左の端 |
| 8 | hem_right | 裾右端 | 身頃の裾ラインの最も右の端 |
| 9 | hem_center | 裾中心 | 裾ラインの中央最下端（垂直方向で最も低い点） |

#### トップスの重要ルール

1. **肩先(1,2) = 袖丈の始点**: 肩幅を測る端点と、袖丈を測る始点は**同じ点**
2. **脇下(3,4)の見つけ方**: 衣類のシルエット（外側の輪郭）を見る。袖が横に張り出し、その後身頃に向かってくびれる。そのくびれの最も内側の点が脇下
3. **肩先 ≠ 袖の端**: 肩先は肩の縫い目の端であり、袖口ではない。袖は肩先から先に続いている
4. **ラグランスリーブの場合**: 肩の縫い目がないため、襟ぐりから袖に向かう斜めの縫い目と、肩のライン（衣類の最上部）が交差する付近をマーキング。判断が難しい場合は visibility=1 に設定

#### トップスの具体例

**セットインスリーブ（普通の袖付け）:**
```
肩先 = 肩の縫い目の端（明確に見える）
```

**ドロップショルダー:**
```
肩先 = 肩の縫い目の端（通常より外側・下側にある）
※ 袖が太い場合、身頃との境目が見えにくいことがある
※ その場合は輪郭の上部で最も角張っている点の付近
```

**ラグランスリーブ:**
```
肩先 = 明確な縫い目がない
→ 襟元から斜めに走る縫い目が、衣類の上部ラインと交わる付近
→ visibility=1 に設定
```

### 2-2. パンツ (pants) — 7点

```
  [0]ウエスト左────[1]ウエスト右
        │              │
        │              │
  [5]ワタリ左─[2]股─[6]ワタリ右
        │              │
        │              │
     [3]裾左        [4]裾右
```

| ID | 名前 | 日本語 | 位置の定義 |
|---|---|---|---|
| 0 | waist_left | ウエスト左端 | ウエストバンド上端の最も左の点 |
| 1 | waist_right | ウエスト右端 | ウエストバンド上端の最も右の点 |
| 2 | crotch | 股交差点 | 内股の縫い目が左右から交わる点（股ぐりの最下点） |
| 3 | left_hem | 左裾 | 左脚の裾の中央下端 |
| 4 | right_hem | 右裾 | 右脚の裾の中央下端 |
| 5 | left_thigh | 左ワタリ | 股交差点(2)と同じ高さ（Y座標）での左脚の最も外側の点 |
| 6 | right_thigh | 右ワタリ | 股交差点(2)と同じ高さ（Y座標）での右脚の最も外側の点 |

#### パンツの重要ルール

1. **股(2)の見つけ方**: 両脚の内股ラインを追うと1点で交わる。その交点
2. **ワタリ(5,6)のY座標**: 股(2)と**同じ高さ**（水平ライン上）で最も外側の点
3. **ゴムウエスト**: ウエストバンドが伸縮する場合、**自然に置いた状態**の幅の端をマーキング（伸ばさない）
4. **ショートパンツ**: 裾(3,4)は脚の開口部の下端中央

### 2-3. スカート (skirt) — 6点

```
  [0]ウエスト左────[1]ウエスト右
        │              │
  [2]ヒップ左────[3]ヒップ右
       ╱                ╲
      ╱                  ╲
  [4]裾左────────[5]裾右
```

| ID | 名前 | 日本語 | 位置の定義 |
|---|---|---|---|
| 0 | waist_left | ウエスト左端 | ウエストバンドの上端の最も左の点 |
| 1 | waist_right | ウエスト右端 | ウエストバンドの上端の最も右の点 |
| 2 | hip_left | ヒップ左 | ウエスト上端から**下に約18cm**の高さでの左端 |
| 3 | hip_right | ヒップ右 | 同じ高さでの右端 |
| 4 | hem_left | 裾左端 | 裾ラインの最も左の点 |
| 5 | hem_right | 裾右端 | 裾ラインの最も右の点 |

#### スカートの重要ルール

1. **ヒップ位置(2,3)**: ウエスト上端から約18cm下。画像上で目測で構いません（正確な距離はAIが算出する）。おおよそウエストの下1/3〜1/2あたり
2. **プリーツスカート**: 裾が波打っている場合、最も左/右に張り出している点をマーキング
3. **ゴムウエスト**: 自然な状態の幅

### 2-4. ワンピース (dress) — 12点

```
          [0] 襟中心
     [1]肩先──────[2]肩先
      │  ＼          ／  │
      │  [3]脇下─[4]脇下 │
      │      │      │    │
   [5]袖口   │      │  [6]袖口
        [10]W左──[11]W右
             │      │
          [7]裾左─[8]裾右
              [9]裾中心
```

| ID | 名前 | 位置の定義 |
|---|---|---|
| 0-9 | トップスと同一 | 上記トップスの定義を参照 |
| 10 | waist_left | くびれ/ウエストラインの左端。ゴムシャーリングがある場合はその位置 |
| 11 | waist_right | くびれ/ウエストラインの右端 |

#### ワンピースの重要ルール

1. **ウエスト(10,11)**: ウエストの切り替えやゴムシャーリングがある場合はその位置。ない場合（ストレートシルエット）は衣類の上端から約40-50%の高さ（自然なくびれ位置）を目測。判断が難しい場合は visibility=1
2. **ノースリーブの場合**: 袖口(5,6)は存在しない → **visibility=0** に設定。肩先(1,2)はアームホール（袖ぐり）の最上端

---

## 3. visibility（可視度）の設定

各キーポイントに visibility 値を設定してください。

| 値 | 意味 | 使う場面 |
|---|---|---|
| **2** | 完全に見える | 通常はこれ |
| **1** | 位置は推測できるが確信がない | ラグランの肩先、布が少し重なっている、縫い目が見えにくい |
| **0** | マーキング不可 / 存在しない | ノースリーブの袖口、A4用紙で完全に隠れている |

**重要**: visibility=0 の場合でも、座標は (0, 0) に設定してください（CVATのデフォルト）。

## 4. よくある判断に迷うケース

### Q1: 肩の縫い目が見えない
A: 衣類の外側輪郭を見てください。上部で最も角張っている点（肩のライン→袖のラインに折れ曲がる点）が肩先です。visibility=1 に設定。

### Q2: 脇下がA4用紙で隠れている
A: A4用紙は衣類の上（中央）に置くため、一部のキーポイントがA4に隠れる場合があります。輪郭から推測してマーキングし、visibility=1に設定。完全に判断できない場合は visibility=0。

### Q3: 袖が身頃に重なっている
A: 袖口(5,6)は見えている位置をマーキング。脇下(3,4)は輪郭のくびれから推測し、visibility=1。

### Q4: パーカーのフードはどうする？
A: フードは無視してください。襟中心(0)はフードの付け根（身頃との接合部分）の中央。

### Q5: 裾が不均一（前後で長さが違う）
A: 前身頃の裾を基準にしてください（前身頃が上に見えている）。

### Q6: ゴムウエストの幅
A: **伸ばさない自然な状態**の端をマーキング。

## 5. 作業手順

1. CVATにログイン → 割り当てられたタスクを開く
2. 画像を表示
3. 衣類のカテゴリを確認（ファイル名の先頭: tops/pants/skirt/dress）
4. 対応するキーポイントテンプレートを選択
5. 各キーポイントを**定義の順番通りに**マーキング
6. 判断に迷う点は visibility=1 に設定
7. 存在しない点は visibility=0 に設定
8. 全点マーキング後、次の画像へ
9. バウンディングボックス（衣類全体を囲む矩形）も設定

### 品質チェックポイント

マーキング後、以下を確認してください:

- [ ] 全てのキーポイントをマーキングしたか
- [ ] 肩先(1,2)は袖の途中ではなく、肩の縫い目の端にあるか
- [ ] 脇下(3,4)は衣類のくびれ（袖→身頃の境目）にあるか
- [ ] 左右対称に近い位置になっているか（衣類が対称なら）
- [ ] ワタリ(5,6)のY座標は股(2)と同じ高さか（パンツ）
- [ ] バウンディングボックスは衣類全体を囲んでいるか

## 6. 納品形式

- CVATからCOCO Keypoint 1.0形式でエクスポート
- ZIPファイルで納品
- 含まれるファイル:
  - `instances_default.json` — アノテーションJSON
  - `images/` — 元画像のコピー（不要であれば画像なしでOK）

## 7. 参考: 各キーポイントと採寸項目の対応

このアノテーションから算出される採寸値:

| 採寸項目 | 計算方法 | 使うキーポイント |
|---|---|---|
| 肩幅 | 左肩先↔右肩先の水平距離 | 1, 2 |
| 袖丈 | 肩先→袖口のユークリッド距離 | 1→5 or 2→6 |
| 身幅 | 左脇下↔右脇下の水平距離 | 3, 4 |
| 着丈 | 襟中心→裾中心の垂直距離 | 0, 9 |
| ウエスト | ウエスト左↔右の水平距離 | 0, 1 (パンツ/スカート) |
| 総丈 | ウエスト上端→裾の垂直距離 | 0→3 (パンツ), 0→4 (スカート) |
| 股下 | 股交差点→裾の垂直距離 | 2, 3 (パンツ) |
| ワタリ | ワタリ左↔右の水平距離 | 5, 6 (パンツ) |
| ヒップ | ヒップ左↔右の水平距離 | 2, 3 (スカート) |
