## 1. 注文確認

```
ご注文の確認についてご連絡いただきありがとうございます。

受付番号「{受付番号}」にてご注文を確認いたしました。
現在、{入金待ち / 発送準備中}の状態です。

【コンビニ払い・銀行振込の場合】
お支払い期限はご注文日より3日間となっております。
期限を過ぎますと自動キャンセルとなりますのでご注意ください。
```

## 2. 入金確認

### 入金済みの場合

```
ご入金の確認についてお問い合わせいただきありがとうございます。

確認いたしましたところ、{日付}付けでご入金を確認済みです。
現在、発送準備を進めておりますので、もうしばらくお待ちください。
```

### 未着の場合

```
ご入金の確認についてお問い合わせいただきありがとうございます。

現時点ではまだご入金の確認が取れておりません。
金融機関によっては反映までに1〜2営業日かかる場合がございます。

お手数ですが、お振込日と金額をお知らせいただけますでしょうか。
改めて確認のうえご連絡いたします。
```

## 3. 発送状況

### 発送済みの場合

```
発送状況についてお問い合わせいただきありがとうございます。

ご注文の商品は{日付}に発送済みでございます。

配送業者：{配送業者名}
伝票番号：{伝票番号}

下記URLより配送状況をご確認いただけます。
{追跡URL}
```

### 未発送の場合

```
発送状況についてお問い合わせいただきありがとうございます。

ご注文の商品は現在発送準備中です。
準備でき次第、発送通知メールをお送りいたしますので
もうしばらくお待ちいただけますと幸いです。
```

## 4. 商品の状態（クレーム）

### 初回対応

```
このたびはご迷惑をおかけし、大変申し訳ございません。

お送りいただいた内容を確認いたしました。
状況を詳しく確認させていただきたいため、
該当商品のお写真をお送りいただけますでしょうか。

確認のうえ、返品・交換等の対応をご案内いたします。
```

### 確認後、返品対応

```
お写真をご確認いたしました。
ご不便をおかけし重ねてお詫び申し上げます。

つきましては、以下の通り返品対応をさせていただきます。

・対象商品：{商品名}
・返送先：{住所}
・返送料：弊社負担（着払いにてお送りください）

商品到着確認後、ご返金の手続きを進めさせていただきます。
```

## 5. キャンセル

### 未発送の場合

```
キャンセルのご連絡をいただきありがとうございます。

受付番号「{受付番号}」のご注文について、
キャンセル処理を承りました。

【クレジットカードの場合】
ご返金はカード会社経由で処理いたします。
反映までに数日〜最大1ヶ月ほどかかる場合がございます。

【コンビニ払い・銀行振込で入金済みの場合】
ご返金先の口座情報をお知らせいただけますでしょうか。
確認後、速やかにお振込みいたします。
```

### 発送済みの場合

```
大変恐れ入りますが、すでに商品を発送済みのため
キャンセルをお受けすることが難しい状況です。

商品到着後の返品対応であれば承れますので、
ご希望の場合はお知らせください。
```

```

大変申し訳ございませんが、ご注文商品の一部に
欠品が発生いたしました。

■ 欠品商品
・{商品名}

欠品商品を除いた内容でご注文を再作成いたしました。
更新後の確認ファイルを下記よりご確認ください。

{確認リンクURL}

■ 変更後のご注文内容
・商品点数：{点数}点
・合計金額：{金額}円（税込）

ご不都合がございましたら、お気軽にお申し付けください。
```

## 7. ログインの問題

```
ログインについてお問い合わせいただきありがとうございます。

パスワードをお忘れの場合は、ログイン画面の
「パスワードを忘れた方はこちら」からリセットが可能です。
ご登録のメールアドレス宛に仮パスワードをお送りいたします。

上記でも解決しない場合は、ご登録のメールアドレスを
お知らせいただければ、こちらで確認いたします。
```

## 8. ポイント・ランク

```
ポイントについてお問い合わせいただきありがとうございます。

現在のお客様の情報は以下の通りです。

・ランク：{レギュラー/シルバー/ゴールド/ダイヤモンド}
・ポイント残高：{残高}ポイント
・ポイント有効期限：最終獲得日より2年間

ポイントは次回ご注文時に1ポイント＝1円としてご利用いただけます。
注文画面にてご利用ポイント数をご入力ください。
```

```
副業で古着販売をされている方を中心にご利用いただいております。

ご利用いただけます。

ご不明な点がございましたら、何でもお気軽にお聞きください。
```
