# メルカリ出品テキスト自動生成ツール

## できること
配布用リスト（xlsx）の商品データから、メルカリ向けの以下を自動生成します：
- **商品タイトル**（40文字以内・検索最適化済み） → E列に書き込み
- **商品説明文**（ブランド紹介・着こなし提案・ハッシュタグ付き） → K列に書き込み

## セットアップ（初回のみ）

### 1. Python 3 をインストール
まだ入っていない場合は https://www.python.org/ からダウンロード

### 2. ライブラリをインストール
このフォルダで以下を実行：
```
pip install -r requirements.txt
```

### 3. APIキーを設定

Mac / Linux の場合：
```
export OPENAI_API_KEY="あなたのAPIキー"
```

Windows の場合：
```
set OPENAI_API_KEY=あなたのAPIキー
```

※既存のGPTで使っているAPIキーと同じものでOKです

## 使い方

### 基本
```
python generate.py ファイル名.xlsx
```
→「ファイル名_generated.xlsx」が出力されます

### テスト（特定の行だけ処理）
```
python generate.py ファイル名.xlsx --rows 3-5
```

### すべて再生成（既存の説明文があっても上書き）
```
python generate.py ファイル名.xlsx --force
```

### ドライラン（APIを呼ばず、入力データだけ確認）
```
python generate.py ファイル名.xlsx --dry-run
```

## 入力ファイルの列構成

| 列 | 内容 | 備考 |
|----|------|------|
| A | 確認 | |
| B | 箱ID | |
| C | 管理番号 | |
| D | ブランド | 生成の入力に使用 |
| E | AIタイトル候補 | → メルカリ用タイトルで上書き |
| F | アイテム | 生成の入力に使用 |
| G | サイズ | 生成の入力に使用 |
| H | 状態 | 生成の入力に使用 |
| I | 傷汚れ詳細 | 生成の入力に使用 |
| J | 採寸情報 | 生成の入力に使用 |
| K | 即出品用説明文 | → 生成した説明文で上書き |
| L | 金額 | |

※1行目はメタ情報、2行目が列ヘッダー、3行目以降がデータ

## 注意事項
- K列に説明文がすでにある行は自動でスキップされます（`--force` で上書き可）
- 処理前にバックアップファイル（_backup.xlsx）が自動作成されます
- APIの料金が発生します（1商品あたり約0.3円、100商品で約30円）
