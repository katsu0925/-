AI採寸シート 使い方

印刷設定: 余白「なし」、倍率「カスタム 100」で印刷

使い方

  1. 1.表面の5cmゲージを手持ちの定規で確認（ずれていたら印刷し直す）
  2. 2.衣類を平置きし、この紙を衣類の上（中央付近）に置く
  3. 3.真上からスマホで撮影（定規がカメラの中心に来るように狙う）
  4. 4.アプリにアップロード → 自動で採寸＋商品説明を生成

撮影のコツ

OK 真上から撮る（カメラと床が平行）
明るい場所で影が出ないように
シワを伸ばし左右対称に整える
定規の両端の黒マークが写っている
衣類全体が画角に入っている
NG 斜めから撮る（遠近で歪む）
フラッシュ使用（反射で黒マークが消える）
暗い場所・影が濃い
定規が衣類の下に隠れている
衣類の端が画面外に切れている

スマホの水平ガイド設定

真上から撮影するために、スマホのカメラに内蔵されている水平ガイド機能をONにしてください。

iPhone 「設定」→「カメラ」→
「グリッド」をON
→ カメラを真下に向けると画面中央に十字マーク（+）が表示されます。2つの+が重なった時が水平です。
Android カメラアプリの「設定」→
「グリッドライン」をON
→ 一部の機種（Galaxy等）は「水平ガイド」もあります。カメラ設定内で「レベル」や「水平」を探してONにしてください。
長持ちさせるコツ
・厚紙（画用紙など）に印刷すると丈夫で繰り返し使えます
・ラミネート加工するとさらに耐久性UP（100均でラミネートフィルムが購入できます）
・紙の裏にマスキングテープを貼るとすべり止めになります

注意
・両端の黒い四角がAIの基準点です。折ったり汚したりしないでください
・コートなど大きい衣類は、十分な距離を取って全体を撮影してください
・採寸結果は「平置き実寸（参考値）」です。±1〜2cmの誤差があります
