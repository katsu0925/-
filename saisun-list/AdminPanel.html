適用順序
FHP(排他) → 数量割引(デタウリのみ) → 会員割引(両チャネル) → クーポン控除 → ポイント利用 → 送料計算
送料: ダイヤモンド無料 → クーポン無料 → ¥10,000以上無料 → テーブル計算
初回全品半額（FHP） -
有効時は他の全割引（数量・会員・クーポン）が無効になります。購入回数0回のログインユーザーのみ適用。
例: 0.50 = 50%OFF
期限を過ぎると自動的に無効になります。空欄の場合は無期限。
数量割引 デタウリ個品のみ
デタウリ個品のみ適用。アソート商品には適用されません。閾値は降順で判定（高い方から順にマッチ）。
会員割引 -
ログイン必須。デタウリ・アソート両方に適用。クーポンのcomboMemberフラグで併用制御。
例: 0.10 = 10%OFF
SNSシェアキャンペーン -
承認時に¥1,000固定クーポンを自動生成。90日有効、1人1回。他割引と非併用。
送料無料閾値・紹介プログラム
