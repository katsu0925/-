## 2. 重要: 販売率50%での収益構造

### 1点あたりの損益（販売率50%）

```
【売れた場合（50%の確率）】
  売上: ¥606（商品¥519 + 送料按分¥87）
  変動費: ¥84（送料実費¥43 + 決済手数料¥22 + 梱包¥19）
  貢献利益: ¥522

【売れなかった場合（50%の確率）】
  売上: ¥0
  変動費: ¥0

【1点あたり期待値（仕入れた全商品の平均）】
  期待売上: ¥606 × 50% = ¥303
  期待変動費: ¥84 × 50% = ¥42
  仕入れ原価: ¥300
  ────────────────────
  1点あたり損益 = ¥303 − ¥42 − ¥300 = −¥39（赤字）
```

> **販売率50%では、倉庫代を考える以前に1点あたり¥39の赤字。**
> 損益分岐の販売率は **58%**（= ¥300 ÷ ¥522）。

## 8. 結論とまとめ

### 販売率50%の現実

| 指標 | 結果 |
|------|------|
| 1点あたり損益 | **−¥39（赤字）** |
| 損益分岐 販売率 | **58%** |
| 仕入れ¥10〜15万での月間損益 | **−¥13,000〜−¥19,500**（倉庫代なしでも赤字） |

> **販売率50%ではビジネスが成立しない。** 仕入れ額を増やしても赤字が拡大するだけ。

### 黒字化の条件

| 条件 | 最低販売率 |
|------|----------|
| 倉庫代なしで黒字 | **58%** |
| 倉庫代¥1万で黒字（仕入れ¥15万） | **62%** |
| 倉庫代¥2万で黒字（仕入れ¥15万） | **66%** |
| 倉庫代¥3万で黒字（仕入れ¥15万） | **70%** |

### 現実的なシナリオ

**販売率70% + 倉庫代¥1万の場合:**
- 仕入れ¥10万 → 手残り **¥11,800/月**
- 仕入れ¥15万 → 手残り **¥22,700/月**

**販売率80% + 倉庫代¥2万の場合:**
- 仕入れ¥10万 → 手残り **¥19,200/月**
- 仕入れ¥15万 → 手残り **¥38,800/月**

## 注意事項

- 平均仕入れ値¥300は仮定値。実際の仕入れ単価により大きく変動
- 会員割引（10%OFF）利用者がいる場合、売上はさらに低下
- 返品・欠品（廃棄）率は別途考慮が必要
- 作業報酬（スタッフ費）は未計上
- 送料は全国平均。実際の顧客分布により変動
- 販売率は新品/ブランド力/季節で大きく変動する
- 未販売在庫は翌月以降に売れる可能性あり（累積在庫による販売率向上の可能性）
