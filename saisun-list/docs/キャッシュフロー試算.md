> 発送構成: 10点発送 70% / 30点発送 30%

## 3. 倉庫代の目安

### 算出式

```
利益 ≒ 仕入れ代 × 0.74 − 倉庫代
```

> 1点あたりの貢献利益 ≒ ¥222（売上¥606 − 原価等¥384）
> 仕入れ ¥300/点 に対して 0.74倍 が倉庫代前の利益

## 6. 旧モデル（全件10点発送）との比較

| 指標 | 旧: 全10点発送 | 新: 70/30混合 | 差分 |
|------|--------------|-------------|------|
| 平均点数/発送 | 10点 | 16点 | +60% |
| 平均販売単価 | ¥550 | ¥519 | −¥31（30点割引影響） |
| 発送回数/月（¥15万） | 50回 | 31回 | −38%（**作業負担減**） |
| 送料実費/月（¥15万） | ¥32,500 | ¥21,500 | −¥11,000（**コスト減**） |
| 倉庫代前利益（¥15万） | ¥96,000 | ¥106,135 | **+¥10,135** |

> **混合モデルのメリット**: 30点大口発送が入ることで発送回数が減り、送料実費・梱包コストが下がる。
> 30点割引（10%OFF）による売上減を上回るコスト削減効果があり、**利益は約1万円改善**。

## 注意事項

- 平均仕入れ値¥300は仮定値。実際の仕入れ単価により大きく変動
- 会員割引（10%OFF）利用者がいる場合、売上はさらに低下
- 返品・欠品（廃棄）率は未考慮（5%発生で利益約5%減少）
- 作業報酬（スタッフ費）は未計上
- 送料は全国平均。実際の顧客分布により変動
- 全商品が同月内に売切れる前提（在庫回転率100%）
